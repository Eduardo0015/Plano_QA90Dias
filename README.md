# Plano QA em 90 Dias

Este repositório documenta minha jornada prática de estudos em QA,
com foco em testes manuais, documentação e ferramentas.

## Estrutura
- bug-reports: registros de bugs documentados
- test-cases: casos de teste escritos durante o plano

## Ferramentas
- Jira
- Git / GitHub
- Postman
- Cypress

## Conteúdo do repositório

/test-cases → planilhas com casos de teste estruturados  
/bug-reports → relatórios de defeitos com passos e evidências  
/screenshots → prints dos testes executados e falhas encontradas  
/Plataforma_Saucedemo → cenários específicos dessa aplicação  
/Plataforma_HerokuApp → testes de API e validação SQL
